# Sales Dashboard

This is a simple Power BI project based on e-commerce sales data.

In this dashboard, I have analyzed sales performance, profit, and customer trends using different visuals.

## What I did

- Calculated total sales, quantity, and profit
- Analyzed sales by category and sub-category
- Compared monthly sales and profit (year-wise)
- Checked sales by region and state
- Analyzed different payment modes and shipping types
- Created a 15-day sales forecast

## Tools Used

- Power BI

## File

- Sales_Forcast.pbix

## How to use

Download the file and open it in Power BI Desktop to explore the dashboard.
